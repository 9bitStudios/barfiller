Barfiller

A simple jQuery plugin that gives you percentage-based animated bar filling...

Download this project as a .zip file Download this project as a tar.gz file

Barfiller.js is a simple jQuery plugin that gives you percentage-based animated bar filling. All you have to do is set the data-percentage attribute to the percentage value that you want and then call Barfiller on the container element.
